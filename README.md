# Martmax - Rwanda's Leading Electronics Store

A premium e-commerce platform built with modern web technologies, featuring a curated selection of electronics, smartphones, gadgets, and accessories with fast delivery across Kigali.

## Features

- **Product Catalog**: Browse 120+ products across multiple categories
- **Advanced Search**: Find products by name, brand, or category
- **Shopping Cart**: Easy-to-use shopping cart with local storage
- **Product Details**: Detailed product information with ratings and reviews
- **Responsive Design**: Optimized for desktop, tablet, and mobile devices
- **Fast Performance**: Optimized images and lazy loading
- **Payment Options**: MTN MoMo, Visa, and Mastercard support
- **Quick View**: Fast product previews without leaving the page

## Getting Started

### Prerequisites
- Any modern web browser (Chrome, Firefox, Safari, Edge)
- No build process or dependencies required

### Running Locally

1. Clone the repository:
```bash
git clone https://github.com/joshthewhite75/Martmax.git
cd Martmax
```

2. Open in your browser:
   - Simply open `index.html` in your browser, or
   - Use a local server (recommended):
   ```bash
   # Python 3
   python -m http.server 8000
   
   # Python 2
   python -m SimpleHTTPServer 8000
   
   # Or use Node.js http-server
   npx http-server
   ```

3. Visit `http://localhost:8000` in your browser
